# Not-To-Do App

やらないことを置いておくための、シンプルなWebアプリ。

生産性を上げるためのツールではありません。
「やらない」と決めることで、自分を少し軽くするための場所です。

---

## 🧠 コンセプト

- 期間は持たない
- 進捗管理もしない
- 完了もない
- ただ、やらないことを並べる

生産性より、余白を。

---

## ✨ 機能

- やらないことを追加
- 削除
- ローカル保存（LocalStorage）

---

## 🛠 技術スタック

- React
- TypeScript
- Vite
- useReducerによる状態管理

---

## 🚀 ローカル起動方法

```bash
npm install
npm run dev
