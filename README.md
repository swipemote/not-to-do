# Not-To-Do

🔗 Live Demo  
https://not-to-do-ashy.vercel.app/

---

## 概要

やらないことを置いておくための、シンプルなWebアプリ。

生産性を上げるためのツールではありません。  
「やらない」と決めることで、少し余白をつくるための場所です。

期間も、進捗も、評価もありません。

---

## コンセプト

- 期間を持たない
- 完了概念を持たない
- タスク管理をしない
- ただ、やらないことを並べる

管理より、手放すことを。

---

## 機能

- やらないことの追加
- 削除
- LocalStorageによる永続化

---

## 技術スタック

- React
- TypeScript
- Vite
- useReducerによる状態管理

---

## 設計について

状態管理には `useReducer` を採用し、  
将来的な拡張を見据えつつも、構造は最小限に抑えています。

不要な状態（完了フラグや期間概念）は持たず、  
コンセプトに沿ってロジックを削ぎ落としました。
